Synopsis du projet :
Etant tous les 3 fans de jeux vidéos, nous voulions creer un site dédié aux gens recherchant de nouveaux jeu a essayer et qui sait peut leur prochain jeu préféré. Vous trouverez sur ce site une liste de jeu (choisis par nos soins selon les gouts et la popularité dans le monde),(bien evidemment tous les jeux existant ne sont pas present pour des questions de temps) parmis lesquels vous pourrez like ceux qui vous interesse. Ces jeux se retrouveront sur votre profil dans un tableau dynamique. Sur le site les jeux sont rangées selon leur catégorie, de cette manière, la recherche est plus simple. Si vous aimez uniquement les jeux d'action, pas besoin de tous les voirs, vous pourrez n'avoir acces qu'a ceux qui remplissent les critères demandés.
